# Subscription Manager

OrderGroove's customer-facing front-end widget where shoppers manage their subscriptions (skip, change date, change product, change frequency, update payment, etc.). Highly themable and embeddable in any storefront.

If a developer is asking about backend / REST API integration only, they probably don't need this directory.
