# Support

## Official Hyvä Support Channels

We try to handle support as transparent as possible, so that anyone can learn from it. That’s why we prefer to answer questions on [Slack](slack-channels.html).

Any sales/license related questions can be asked via email or our [contact form](https://www.hyva.io/contact).

Customers with a license for a commercial Hyvä product can submit technical support questions via their account dashboard on [www.hyva.io/request-hyva-support/request](https://www.hyva.io/request-hyva-support/request/index/).

Technical support questions asked by via email will receive a standard reply redirecting to technical support form and Slack community.

## Hyvä Products - Private GitLab Access

The Hyvä commercial product sources on private GitLab are accessible for **technical partners only** at [gitlab.hyva.io/hyva-themes/](https://gitlab.hyva.io/hyva-themes/).

GitLab is not a support channel. Please report issues using our technical support form (for paid license holders) or Slack, to be confirmed by the Hyvä team.

## Hyvä Theme - Public GitHub

Since release 1.4.0 Hyvä Theme source code is publicly available in repositories at [github.com/hyva-themes/magento2-default-theme](https://github.com/hyva-themes/).

Please note, GitHub is not a support channel. Support for Hyvä Theme is available to customers with a paid license via the [technical support form](https://www.hyva.io/request-hyva-support/request/index/) in the customer dashboard.
Community support is available through the Hyvä Slack community.

## React Checkout - GitHub

The Hyvä React Checkout is a built and maintained by our community, mainly by [Rajeev K Tomy](https://github.com/rajeev-k-tomy) partly supported by [integer\_net](https://integer-net.com).
Please note that it is not a Hyvä product, and support is offered for this (free) extension.
Please use the [GitHub issues on the repository](https://github.com/hyva-themes/magento2-react-checkout/issues) to communicate issues to the maintainer.
