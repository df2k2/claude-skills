# Hyvä Checkout vs React Checkout

Hyvä Checkout and the React Checkout are two completely separate checkout solutions for Magento 2 with Hyvä Themes. Despite the similar names, the two checkouts share no codebase, no architecture, and no development team. This page clarifies the differences between Hyvä Checkout and the React Checkout to help you choose the right solution.

## Hyvä Checkout (Commercial Product)

[Hyvä Checkout](https://www.hyva.io/hyva-checkout.html) is a commercial checkout built and actively maintained by Hyvä Themes. Hyvä Checkout uses Magewire (a Laravel Livewire port for Magento) to deliver a server-rendered, Alpine.js-driven checkout experience. Hyvä Checkout receives regular updates, bug fixes, and official support through Hyvä.

## React Checkout (Open-Source Community Project)

The [React Checkout](https://github.com/hyva-themes/magento2-react-checkout) is a community-maintained, open-source checkout that uses React on the frontend. The React Checkout does not receive official support from Hyvä Themes and relies entirely on community contributions for maintenance and new features.

## Why the Confusion?

The naming confusion between Hyvä Checkout and the React Checkout has a historical reason. When the React Checkout was first released, it was originally named "Hyvä Checkout." Some third-party integration providers also used that original name in their documentation and module listings. The React Checkout has since been renamed, but older references to the original name still appear in some places.

Quick Rule of Thumb

If a third-party module or blog post mentions "Hyvä Checkout" but references React components or a React-based frontend, it is referring to the **React Checkout**, not the commercial **Hyvä Checkout** product.

## Feature Comparison at a Glance

| Aspect | Hyvä Checkout | React Checkout |
| --- | --- | --- |
| **Type** | Commercial product | Open-source community project |
| **Maintained by** | Hyvä Themes (official) | Community contributors |
| **Frontend technology** | Alpine.js + Magewire | React |
| **Official support** | Yes | No |
| **Active development** | Yes | Community-driven |
