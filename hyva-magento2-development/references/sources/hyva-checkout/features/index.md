# Introduction

The purpose of this section of documentation is to provide merchants with a high-level overview of the features of the
Hyvä Checkout and developers with some technical details of particular implementations. The hope is to ensure merchants
understand the consequences of tweaking checkout configuration options in their admin area and empower developers to
make their customisations, extensions, compatibilities etc.
