# Upgrading to Hyvä Commerce 0.5.1

This is a feature release for initial versions of Image Editor and Admin Theme.

## Notable news

### Image Editor

The image editor adds the ability to edit images directly within the existing media gallery, offering tools to adjust images (crop, rotate, flip, resize), apply filters, fine-tune (brightness, hue, etc.) and add annotations. It also allows for images to be saved as duplicates, and includes the ability to restore the original image. It is also fully compatible with Hyvä CMS and Page Builder.

### Admin Theme

The Admin Theme is based on the [Mage-OS M317 Admin Theme](https://github.com/mage-os-lab/theme-adminhtml-m137) and gives the admin panel a modern look and feel. In addition, configuration is provided to select the active theme (in case you want to revert to the defaul) and offers the ability to upload custom logos and favicons.
