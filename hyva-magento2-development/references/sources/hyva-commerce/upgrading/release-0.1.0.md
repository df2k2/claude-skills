# Hyvä Commerce 0.1.0 (Initial Early Access Release)

Release 0.1.0 includes our initial version of Hyvä CMS, along with our base `Hyva_Commerce` module that is used for shared code across all Hyvä Commerce functionality.

## Notable news

### Hyvä CMS

This initial release includes:

- Replacement CMS for Page Builder with our new Liveview editor
- View and update content in real-time with a live storefront preview
- Create, drafts, share with colleagues, and only publish when ready
- Full version history with the ability to preview and restore any revision
- A wide range of components, including many built from Hyvä UI components
- Ability to modify and create your own custom components
- Support for adding content to CMS Pages and Blocks
- Editor features including: undo/redo, cut/copy/paste and import/export
