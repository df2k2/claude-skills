# Upgrading to Hyvä CMS 1.2.0

This version is not yet released

1.2.0 version is not yet released. This is currently in beta. The beta version is available [here](changelog-beta.html#hyva-cms-120-beta1-2026-02-27).

This update introduces support for Attribute and Tailwind v4 support for Hyvä CMS.

You can upgrade to the beta version by running the following command:

```
composer require hyva-themes/commerce-module-cms:1.2.0-beta1
```

If you have installed the metapackage, you can upgrade to the beta version by running the following command:

```
composer require hyva-themes/commerce-module-cms:"1.2.0-beta1 as 1.1.1"
```

**Note:** This assumes you are on the latest version of the metapackage.
