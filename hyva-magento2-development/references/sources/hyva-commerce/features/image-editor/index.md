# Image Editor

Our Image Editor, included with [Hyvä Commerce](https://www.hyva.io/hyva-commerce.html), is an easy-to-use tool integrated into the Magento Admin. It allows content creators to perform basic image transformations such as resizing, cropping, flipping, fine-tuning, and applying various filters. It supports `JPEG (JPG)` and `PNG` image types and can be used anywhere the Media Gallery is utilised, including within [Hyvä CMS](../cms/index.html).

Image Editor is built upon a forked version of [Filerobot Image Editor](https://github.com/scaleflex/filerobot-image-editor).
