# Admin Dashboard

Admin Dashboard is a revitalisation of the Magento Admin Panel's landing page, offering a range of widgets that allow
admin users to create personalised dashboards with content relevant to them.

Merchants and store administrators can get started with our range of [user guides](user-guides/index.html), while
developers can learn more through our [developer documentation](devdocs/index.html).
