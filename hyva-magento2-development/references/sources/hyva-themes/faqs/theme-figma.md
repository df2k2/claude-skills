# Is there a Figma file for the default theme?

There is no official Figma file for the default theme, only for the [Hyvä UI Library](../../hyva-ui-library/figma.html).

However, thanks to Kristaps Rjabovs from Magebit, a [Figma file based on an older version of the default theme](https://www.figma.com/file/Lpvz6rU4Ruwb5bBMJRU5TY/Hyva-Template?type=design&node-id=0%3A1&mode=design&t=yc2Ia9iHcEViDT1N-1) is available publicly.

While extensive, this implementation isn't too up-to-date with changes made to the default theme over time, and the file is for desktop only, without the responsive behavior.

It shows a lot of effort was put into it, and we appreciate it being shared as a free resource!
