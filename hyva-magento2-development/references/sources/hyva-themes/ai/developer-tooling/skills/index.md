# Hyvä AI Skills

Hyvä AI Skills provides AI-powered skills for Magento 2 development with Hyvä Theme. These skills extend AI coding assistants with specialized knowledge for creating Hyvä themes, modules, and CMS components.

For more information see our [blog announcement article](https://www.hyva.io/blog/news/hyva-ai-skills.html) and the [GitHub repository](https://github.com/hyva-themes/hyva-ai-tools) containg the skill files, resources, and installation instructions.
