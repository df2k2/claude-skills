# 4. Working with Alpine.js

## Versions in Hyvä

The Hyvä Default Theme uses the following Alpine.js versions in the theme:

- Alpine.js v2.x in Default Theme v1.0.x / v1.1.x
- Alpine.js v3.x in Default Theme v1.2.x / v1.3.x

It is possible to use both v2 and v3 based Hyvä themes for different store views,
but it is not possible to use v2 and v3 within one theme.

## Alpine.js documentation

The main Alpine documentation site [alpinejs.dev](https://alpinejs.dev) was launched with the release of Alpine V3 features and therefore only contains documentation on version 3.
To visit the V2 docs please refer to the documentation on GitHub: [github.com/alpinejs/alpine/tree/v2.8.2#readme](https://github.com/alpinejs/alpine/tree/v2.8.2#readme)

## Resources

The best introduction we know is the readme linked above.
However, since tastes are different, here are some other resources to get started with Alpine.js:

### General

- A list of Alpine resources: [github.com/alpine-collective/awesome](https://github.com/alpine-collective/awesome)
- A weekly Alpine focused newsletter: [alpinejs.codewithhugo.com/newsletter](https://alpinejs.codewithhugo.com/newsletter/)

### Learning

- Some videos which are a great introduction to Alpine: [laracasts.com/series/alpine-essentials/](https://laracasts.com/series/alpine-essentials/)
- Video series which goes under the hood to explain how Alpine works: [laracasts.com/series/building-alpinejs](https://laracasts.com/series/building-alpinejs)
- Resources and Examples for Alpine.js: [www.alpinetoolbox.com](https://www.alpinetoolbox.com/)

### Debugging

- Alpine Debugger: [alpinejs.pro](https://alpinejs.pro/)
