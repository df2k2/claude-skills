# Editor Setup

## Alpine.js IntelliSense

PHPStormVSCodeZed

Install the [Alpine.js Support plugin](https://plugins.jetbrains.com/plugin/15251-alpine-js-support) from the JetBrains Marketplace.

For configuration options, see the plugin documentation.

Browse and install one of the available [Alpine.js plugins for VS Code](https://marketplace.visualstudio.com/search?term=tag%3Aalpinejs&target=VSCode&category=All%20categories&sortBy=UpdatedDate).

For configuration options, see the documentation for the plugin you choose.

Alternatively, you can create your own code snippets. The Laravel News blog has a good walkthrough at [laravel-news.com/vs-code-snippets-for-livewire-and-alpinejs](https://laravel-news.com/vs-code-snippets-for-livewire-and-alpinejs).

Browse and install one of the available [Alpine.js extensions for Zed](https://zed.dev/extensions?query=alpinejs).

For configuration options, see the documentation for the extension you choose.

Alternatively, you can create your own code snippets. See the [Zed docs on snippets](https://zed.dev/docs/snippets) for how to set them up.

*If you have suggestions or your own solution for a specific editor, feel free to share it with us on Slack.*
