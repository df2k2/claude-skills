# Updating Tailwind CSS

The Tailwind CSS upgrade guide you should follow depends on your Hyvä theme version.

- For themes older than version 1.2, please see [Updating to Tailwind v3](../working-with-tailwindcss/updating-to-tailwind-3.html).
- For themes older than version 1.4, please see [Updating to Tailwind v4](../working-with-tailwindcss/updating-to-tailwind-4.html).
