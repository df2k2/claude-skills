# Upgrading to 1.4.3

Release 1.4.3 is a small maintenance and bugfix release.

Even if not updating the Default Theme to 1.4.3, it is safe to update the `Hyva_Theme` module to the latest version (package `hyva-themes/magento2-theme-module`).

## Notable news

- CSP Default Theme: fix display of Taxes on the Cart page
- Fixed
- Regular Theme: fix PayPal In-Context buttons race condition that sometimes prevented initialization
