# Editor Setup

## Tailwind CSS IntelliSense

PHPStormVSCodeZed

Tailwind CSS autocomplete is built into all JetBrains IDEs, including PhpStorm — no plugin required.

Note that not all features from the official VS Code plugin are available. For full configuration options, see the [PhpStorm docs on Tailwind CSS](https://www.jetbrains.com/help/phpstorm/tailwind-css.html).

Install the official [Tailwind CSS IntelliSense VS Code plugin](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss) for autocomplete, linting, and hover previews.

For full configuration options, see the plugin documentation.

Zed supports Tailwind CSS out of the box — no setup needed.

For details on what's supported, see the [Zed docs on Tailwind CSS](https://zed.dev/docs/languages/tailwindcss).

*If you have suggestions or your own solution for a specific editor, feel free to share it with us on Slack.*
