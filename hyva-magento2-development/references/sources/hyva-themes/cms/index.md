# CMS Content and Hyvä

In a flexible system like Magento there are many ways to use CMS content.

The most integrated way to manage content in Hyvä is [Hyvä CMS](../../hyva-commerce/features/cms/index.html), which is part of [Hyvä Commerce](../../hyva-commerce/index.html). This offers many benefits, both for merchants and developers, over the existing inbuilt Page Builder solution, as well as external solutions. For more details, and how to get a license, see the [Hyvä Commerce page](https://www.hyva.io/hyva-commerce.html) on our website and our [Hyvä CMS docs](../../hyva-commerce/features/cms/index.html).

There also are integrations for third-party services like [Prismic](https://github.com/elgentos/magento2-prismicio) or [Storyblok](https://www.slideshare.net/mzeis/headless-cms-for-magento-using-hyv-and-storyblok),
while for some projects the built-in CMS (static) Pages and Blocks, PageBuilder and [Widgets](../../hyva-widgets/introduction-video.html) are sufficient.
