# Patterns

Code repeats itself.

Not exactly, but the general shape of it.

In Luma things are done in a certain way. Some common practices have been established over time to solve specific problems.

You now need to solve the same kind of problems in Hyvä.

This section collects common transformations from Luma patterns into Hyvä patterns. Knowing them will make you a lot faster.

This collection is not complete and probably never will be, but we will try to expand it as we recognize patterns ourselves.
