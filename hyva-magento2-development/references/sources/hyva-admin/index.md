# Overview

## About

Hyva\_Admin is a Magento 2 module that offers a new way to create admin grids.

Its goal is to improve developer experience and effectiveness when creating new admin grids.

Existing grids and forms are not affected by installing the module; they remain unchanged.

Hyva\_Admin exists to improve the developer experience while creating new grids.

In future form creation will be supported, too.

## Installation

The module is open source and can be found at the public [GitHub repository](https://github.com/hyva-themes/magento2-hyva-admin/).
