# How can I contribute to Hyvä UI?

For general questions about contributing, refer to the [Hyvä Themes FAQ on how to contribute](../../hyva-themes/faqs/hyva-contribution-guideline.html) for general questions.

## Suggesting a New UI Component

We're always looking for ways to improve Hyvä UI.

Feel free to share your suggestions in the Slack **#hyva-ui** channel or directly in the GitLab repository.
We'll add them to our UI development backlog.

Note

While we appreciate your code contributions with merge requests (MRs) for new UI components, we typically prioritize designs created through our internal design process to ensure consistency. However, we'll definitely consider your suggestion when working on the component.
