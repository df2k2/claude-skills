# Component Badges

Hyvä UI component README files and the Figma design include badges that give you a quick overview of licensing, design availability, and CMS compatibility.

## Badge Overview

| Badge | Description |
| --- | --- |
| license | Indicates the component is available under the Hyvä UI commercial license |
| figma | Indicates the component is included in the Hyvä UI Figma design file |
| wysiwyg-support | This component can be used for both WYSIWYG pages and theme development |
| wysiwyg-only | This component is specifically designed for use within Hyvä WYSIWYG pages and blocks |
| hyva-cms | This component is included with the Hyvä CMS module |

## License

![license](https://img.shields.io/badge/License-004d32?style=for-the-badge)

The License badge indicates that a component is part of the Hyvä UI library and is governed by the Hyvä UI commercial license.

Access to Hyvä UI requires an active license. For licensing information, visit [hyva.io](https://www.hyva.io).

## Figma

![figma](https://img.shields.io/badge/Figma-gray?style=for-the-badge&logo=Figma)

The Figma badge indicates that the component has a corresponding design available in the Hyvä UI Figma design file.

You can use this design for customization and prototyping. For more information on working with the Figma file, see the [Figma](../figma.html) page.

## Wysiwyg Support

![wysiwyg-support](https://img.shields.io/badge/wysiwyg_support-ffc803?style=for-the-badge)

The Wysiwyg Support badge indicates that a component can be used in two ways.

- As part of your theme, placed using layout XML or `.phtml` templates.
- Inside a WYSIWYG editor, such as Magento PageBuilder or [Hyvä CMS](../../hyva-commerce/features/cms/index.html), using plain HTML.

## Wysiwyg Only

![wysiwyg-only](https://img.shields.io/badge/wysiwyg_only-5ee8ca?style=for-the-badge)

The Wysiwyg Only badge indicates that a component is designed exclusively for use within WYSIWYG editors, such as Magento PageBuilder or [Hyvä CMS](../../hyva-commerce/features/cms/index.html).

These components rely on plain HTML and do not require theme integration.

## Hyvä CMS

![hyva-cms](https://img.shields.io/badge/Hyv%C3%A4_CMS-109c85?style=for-the-badge)

The Hyvä CMS badge indicates that a component is bundled with the Hyvä CMS module. Installing Hyvä CMS provides access to these components without any additional setup.

For more information, see the [Hyvä CMS documentation](../../hyva-commerce/features/cms/index.html).
