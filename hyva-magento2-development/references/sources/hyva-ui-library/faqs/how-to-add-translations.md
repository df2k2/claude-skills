# Translating Hyvä UI Components

The `i18n/en-US.csv` file is available for creating custom translations starting from Hyvä UI version 2.4.0 and onwards.

Hyvä UI offers a powerful and flexible approach to building user interfaces for your Magento 2 store.

However, the UI components themselves don't come with pre-built translations for different languages out of the box.

This is because Hyvä UI components are constantly evolving with new features and improvements.
Maintaining translations for every component in every language with each update would be a significant effort.

**Solution: Customizable Translations with `i18n/en-US.csv`**

Instead, Hyvä UI provides an `i18n/en-US.csv` file within your Magento 2 installation.
This file contains translatable text used throughout the UI components, but only for generic text and not sample text.

You can use this file as a starting point and reference to identify the text that needs to be translated.
