# What is the Hyvä UI Library?

Hyvä UI is a Stand-Alone Product

As of November 2025, with the Hyvä Theme becoming open source, Hyvä UI is a stand-alone commercial product.

Licenses purchased before November 10, 2025, include Hyvä UI. For those using the open-source version of the Hyvä Theme, a separate license is required.

The **Hyvä UI Library** is a collection of pre-built, ready-to-use components designed to accelerate your development process with Hyvä Themes.

These components can be easily customized and seamlessly integrated into your theme, saving you valuable time and effort.

## Beyond a Standard UI Library

Hyvä UI is more than a typical CSS and HTML library. It offers a more integrated and functional approach to UI development.

Each component is crafted for the Hyvä environment and includes fully functional `.phtml` files with integrated logic. This means that when you integrate a Hyvä UI component, it's immediately functional.

You can leverage these pre-built components to enhance your Magento store's frontend without needing to develop the underlying functionality from scratch.

> Consider Hyvä UI as the functional equivalent of Tailwind UI,
> but tailored specifically for Hyvä Themes and with the added benefit of built-in logic.
